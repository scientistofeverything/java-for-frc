# Java Programming for FIRST Robotics Competition
Created by Noelle H., FRC 9709 Isotope

Hi! Welcome to Java Programming for FIRST Robotics Competition, a textbook/guide for programming FRC robots. The textbook consists of two books:
 1. **Fundamentals of Programming in Java for FRC**: A guide to Java basics with a focus on skills necessary for FRC programming.
 2. **Robot Programming in Java for FRC**: A guide for FRC-specific Java programming skills.
