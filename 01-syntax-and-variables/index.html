Unit 1: Syntax and Variables

Terms to know

  • boolean (n.) - a value that can be either true or false
  • char (n.) - a value containing a single character (ex: a letter, digit, or symbol)
  • constant (n.) - a variable that cannot be changed after being defined
  • double (n.) - a a value containing a decimal number
  • int (n.) - a value containing a positive or negative whole number
  • object (n.) - a value that contains multiple data elements
  • primitive (n.) - a value that stores pure data
  • string (n.) - an value that stores a piece of text as a list of characters
  • stack trace (n.) - a message describing the status of a program reported when an problem occurs
  • syntax (n.) - the rules of how different keywords, names, symbols, and values are used to create a program in a certain programming language
  • type (n.) - a certain kind of value
  • variable (n.) - a container for a certain value that is stored in a computer's memory
