Topic 01.01: Compilation and Execution

Java is a programming language. As a programming language, it's a language that people use to write instructions for machines.

Java is a compiled language, meaning that programs written in Java are turned into a format, called bytecode, that's easier for machines to understand.

The process of turning Java code into bytecode is called compilation, while the process of the machine following the program's instructions is called execution.

In order to program in Java, you need to install the Java Development Kit (JDK), a set of tools that lets your computer run Java code.

This book uses Visual Studio Code, an editor for programmers. To use it for Java, you need to install the Java Extension Pack.

Try creating a new file in VS Code with Ctrl+N, and pasting the following code into it:

public class Main {
    public static void main (String[] args) {
        System.out.println("Hello world!");
    }
}

Then save your file as 'Main.java' with Ctrl+S, and click the Run button.

You should see the following output:

Hello world!

Congratulations! You just ran your first Java program.
